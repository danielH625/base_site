Base 32 is a numeral system that encodes binary data into a set of 32 alphanumeric characters, often including A-Z and 2-7. This system is case-insensitive and avoids visually similar characters (e.g., O and 0).

Applications

  • Base 32 is used in QR codes, URLs, and various data storage mechanisms.
  • It plays a role in secure systems like Google Authenticator (time-based one-time passwords).

Why Base 32?

Base 32 strikes a balance between efficiency and readability, making it ideal for situations where human interaction with encoded data is expected.
